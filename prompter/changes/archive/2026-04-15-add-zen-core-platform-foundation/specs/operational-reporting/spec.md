## ADDED Requirements
### Requirement: Consolidated Operational Dashboards
The system SHALL provide internal dashboards and reporting views for enrollment, attendance, staffing, billing, and academic readiness workflows.

#### Scenario: Review school operations from a consolidated dashboard
- **WHEN** an authorized user opens an operational dashboard
- **THEN** the system shows current summary data for the supported operational domains that user is allowed to access

### Requirement: Exportable Tables and Printable Documents
The system SHALL support exportable operational tables and printable academic documents from internal workflows.

#### Scenario: Export or print workflow outputs
- **WHEN** an authorized user requests an export or print action from a supported workflow
- **THEN** the system generates the requested file or print-ready view for the selected data set or academic document
